Телефонный справочник

API:

Получить всех юзеров - api/users

Поиск юзеров по фио - api/user?fio=X

Поиск юзеров по id - api/user?id=X

Поиск юзеров по отделу - api/user?id=department

Удалить юзера по id - api/user?id=X (Method: DELETE)

Получить все отделы - api/departments

Поиск отдела по имени - api/department?name=X

Поиск отдела по id - api/department?id=X

Удалить отдел по id - api/department?id=X (Method: DELETE)

Веб-клиент:

Добавить юзера - add/user

Добавить отдел - add/department

Получить информацию об отделе - get/department?id=X

Получить информацию о юзере - get/user?id=X
